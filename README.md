# 🐱 猫品种智能分类系统

基于YOLOv8和EfficientNet的猫品种识别系统，支持12种猫品种的分类。

## 功能特点
- 🎯 自动检测图片中的猫
- 🔍 识别12种常见猫品种
- 📊 可视化预测结果
- 📱 支持手机端访问

## 使用方法
1. 上传猫的图片
2. 点击"识别猫品种"按钮
3. 查看识别结果和置信度

## 支持的品种
- 阿比西尼亚猫
- 孟加拉猫
- 伯曼猫
- 孟买猫
- 英国短毛猫
- 埃及猫
- 缅因猫
- 波斯猫
- 布偶猫
- 俄罗斯蓝猫
- 暹罗猫
- 无毛猫

## 技术栈
- YOLOv8: 目标检测
- EfficientNetB0: 品种分类
- Streamlit: Web界面
- PyTorch: 深度学习框架
